Grid

Grid

ISLE uses the react-bootstrap grid system, see the associated documentation. The only difference is that the Col component is exported as Column in ISLE.

Example

<Grid>
    <Row>
        <Column sm={6} md={3}>First</Column>
        <Column sm={6} md={3}>Second</Column>
        <Column sm={6} md={3}>Third</Column>
        <Column sm={6} md={3}>Fourth</Column>
    </Row>
</Grid>
