Dashboard

Dashboard

Example:

<Dashboard title="Alerter" onGenerate={(x)=>{ alert(x) }}>
    <SliderInput
        legend="Input value"
    />
</Dashboard>

Options:

  • onGenerate:
  • title: string with a title for the dashboard to be displayed at its top. Default: ''.
  • description: subtitle string displayed beneath the title. Default: ''.
